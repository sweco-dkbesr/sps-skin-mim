The MIM Skin
--------------------

INSTALLATION
------------
Copy this module to /config/modules/custom/skins directory.
Add to one of your modules xml files as so:

<module name="mim" dir="custom/skins/kdi" permissionlevel="public" />

DEVELOPMENT
--------------------
