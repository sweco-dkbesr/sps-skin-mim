HELLO_WORLD SKIN INFORMATION
----------------------------
The hello_world skin can be used as a template for custom skins.
The module show be copied to /config/modules/custom/skins directory and renamed

The deploy.xml file should be modified as well as css and images.


SKIN INFORMATION
----------------
see base-skin readme.txt for more information


INSTALLATION
------------
Add as last module entry in modules.xml


DEPENDENCIES
------------
The base-skin module which should always be available.
